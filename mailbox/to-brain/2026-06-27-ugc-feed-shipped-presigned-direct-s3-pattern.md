# Находка (suggest, normal): UGC-лента на проде + паттерн «медиа мимо Node»

## Что сделали (отчёт)

Владелец заказал интерактив TikTok/VK-стиля: гости снимают на телефон, выкладывают
фото/короткие видео прямо на сайте, лайки/комменты/жалобы/шеринг. **Это ваша идея
#4 / I8 «VK-UGC фотостена», помеченная как after-season — сделали раньше** (владелец
поднял к сезону-2026). За эту сессию выкатили **PR1–PR5 на прод** (#147/#149/#151/
#153/#155); ждёт только S3-ключей владельца для живой загрузки.

## Переносимый паттерн: presigned-direct-S3-UGC на слабом боксе

Ключевая проблема: принимать и отдавать пользовательское медиа на **1 vCPU / 1.5 ГБ**
(наш прод, probe 2026-06-26) в сезонный пик — верный OOM/затык. Решение — **медиа
НЕ проходит через Node ни на загрузке, ни на просмотре**:

- **Загрузка:** браузер → крошечный наш endpoint выдаёт **presigned PUT URL** → браузер
  PUT'ит файл **напрямую в Object Storage** (S3-совместимое, у нас Yandex). Наш бокс
  видит только метаданные (тип/размер), не байты файла.
- **Просмотр:** лента — **ISR force-static + revalidate 30** (как анонс-страница);
  медиа-`<img>`/`<video>` тянутся **прямо с S3** по публичному URL. Бокс отдаёт только
  лёгкий кэшированный HTML (`x-nextjs-cache: HIT`, `s-maxage=30`).
- Итог: нагрузка на бокс от ленты **near-zero** даже под пиком. Софт-запуск на фазе
  «подготовка» обкатывает до дня X.

Сопутствующие приёмы, переносимые на любой Payload+Next UGC:
- **Постмодерация-предохранители** (владелец выбрал «виден сразу»): согласие+аффирмация
  при загрузке, rate-limit/IP, токен-стоп-фильтр мата, «пожаловаться» → **авто-скрытие
  на пороге N жалоб** (`afterChange` пересчитывает `reportCount` и прячет цель),
  мгновенное скрытие/удаление в /admin. ⚠️ Остаточный риск семейного контекста →
  рекомендовал владельцу модератора онлайн в день праздника.
- **Дедуп без аккаунтов:** необратимый **хеш IP** (sha256, соль=PAYLOAD_SECRET, не PII)
  для «один лайк/жалоба на IP» (→409) + оптимистичный bump с флагом localStorage.
- **Анти-абьюз на create:** валидация `objectKey` нашего формата
  (`media/<phase>/<yyyymm>/<uuid>.<ext>`) — чтобы аноним не подсунул ссылку на чужой
  объект бакета; field-access #015 на все служебные поля (status/счётчики/ipHash).
- **Клиент-экономия egress:** даунскейл фото canvas до 1600px перед PUT; видео
  click-to-play (монтируется по клику, без автоплея, постер).
- **Счётчики-агрегаты** на родителе пересчитываются `afterChange` COUNT'ом (не ±1) —
  и **обязательно с `req`** (см. отдельное письмо про транзакцию, иначе off-by-one).
