## Сквозные правила (соблюдать в каждом PR из этого плана)

- **🔒 152-ФЗ.** Любая новая форма с контактами (розыгрыш, регистрация на состязания, подписка) → галка согласия + ссылка на «Политику ПДн» + **серверный authz #015** (read закрыт на персонал) + rate-limit. Анонимные опросы и «Моя программа» PII не собирают → проще.
- **📦 Медиа/трафик.** Больше фото = больше трафика на 1.5 ГБ VPS в пик. Лечится size-вариантами `Media` + lazy-load; тяжёлое (большие галереи, UGC) → **внешнее хранилище** ([ADR-0001](../adr/0001-media-external-storage.md)).
- **🧱 Схема.** Новые коллекции/поля → миграции **#017** (push-inspect + накат ДО мержа из CI, см. handoff §«#017»).
- **⚡ ISR vs live.** Countdown / опросы / «сейчас на майдане» — клиентские компоненты или короткий `revalidate`, не ломая ISR-60.
- **🎨 Единый стиль.** Переиспользовать существующие токены/узоры/компоненты, не плодить новый визуальный язык.
- **PR-flow.** Ветка → PR → diff → OK владельца → squash-merge (ADR-0002).

## C. Интерактив и вовлечение

- ☐ **I1 — ⏳ Обратный отсчёт.** Таймер до открытия на герое. Клиентский компонент, дата из глобала `festival-settings`; не ломает ISR. ⚙️⏳ **S**
- ☐ **I2 — ⭐ «Моя программа».** Отметить события «звездой», личный список. `localStorage` — **без БД и без PII**, ISR-safe; расширение `ScheduleList`. ⚙️ **S-M**
- ☐ **I3 — 📊 Опросы/голосования.** «Любимое состязание», «лучшее подворье», «оцените фестиваль». Коллекции `Poll`+`PollVote`; **анонимно → проще по 152-ФЗ**; анти-накрутка `rateLimit.ts` (cookie+IP); результаты live. ⚙️🔒(лайт) **M**
- ☐ **I4 — 🎁 Розыгрыш призов.** Контакт+согласие → рандом-выбор победителя в админке в день праздника. `Raffle`+`RaffleEntry`; 🔒 согласие + read на персонал (#015) + rate-limit + миграция #017. 🔒⚙️ **M-L**
- ☐ **I5 — 🎟️ Регистрация на состязания.** «Заявиться на куреш / скачки / детские игры». Расширение формы заявок (поле `competitionType`); тот же 152-ФЗ-гейт, что уже работает. 🔒⚙️ **M**
- ☐ **I6 — ✉️ Подписка на анонс.** «Напомнить о Сабантуе» — письмо до праздника (SMTP уже готов, PR #24). Коллекция `Subscribers`+согласие. 🔒⚙️ **S-M**
- ☐ **I7 — 🗺️ Интерактивная карта.** Фильтр пинов по типам (показать/скрыть «еда/сцена/туалеты»), опц. «вы здесь». Расширение `/map`. ⚙️ **M**
- ☐ **I8 — 🖼️ Фотостена гостей (UGC).** Фото от гостей (модерация) или импорт хэштега из VK. **После сезона** — это бэклог #4; обязательны модерация + внешнее хранилище. 🔒⚙️ **L**
- ☐ **I9 — 🔗 Шаринг + OG-картинки.** Динамические OG-превью при репосте события/афиши (Next OG-image). ⚙️ **S-M**
- ☐ **I10 — 📱 PWA + офлайн-программа.** «Добавить на экран», программа/карта без сети (на майдане часто плохой интернет). ⚙️ **M**
- ☐ **I11 — 🌐 Тумблер RU/TT.** Публичное переключение языка — когда появятся реальные TT-переводы (уже в бэклоге handoff). ⚙️ **M**
